This is a simple REST API for managing Posts using Node.js, Express, and MongoDB.

Each Post has:

title – String

content – String

createdAt – Date (auto-generated)


Base URL

  http://localhost:4000/api

Endpoints
1️. Get All Posts
    GET /posts
    
2. Get Single Post
  GET /posts/:id

4. Create a Post
  POST /posts

4.Update a Post
  PUT /posts/:id
  
6. Delete a Post
  DELETE /posts/:id
